The Dirichlet distribution can be parameterized using either a single parameter that applies to each category, or you can specify a different k parameter for each category. In any case, the posterior distribution for a multinomial-Dirichlet model is Dirichlet( + n) , where is a vector of either all the same number, or the hyperparameter choice per category, and n is a vector of the counts of each category.

eth_data <- list(
  N1 = length(data_eth_A),

  y1 = data_eth_A,

stan_model1 <- "
data {
  int<lower=0> N1;
  int<lower=0> N2;
  real y1[N1];
  real y2[N2];
}
parameters {
  real mu1;
  real mu2;
  real<lower=0> sigma1;
  real<lower=0> sigma2;
}
transformed parameters {
  real diff;
  diff = mu1 - mu2;
}
model {
  mu1 ~ normal(0, 100);
  mu2 ~ normal(0, 100);
  sigma1 ~ cauchy(0, 50);
  sigma2 ~ cauchy(0, 50);
  
  y1 ~ normal(mu1, sigma1);
  y2 ~ normal(mu2, sigma2);
}
"
